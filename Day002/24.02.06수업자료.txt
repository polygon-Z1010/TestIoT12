C 프로그래밍


nano 에디터
> apt get install nano
^ : 컨트롤

gcc 옵션
-c : 오브젝트 파일을 만든다.
-o (만드려는 실행파일 이름) (오브젝트 파일이름, 확장자 포함) : 오브젝트 파일을 바로 실행파일로 만든다.


./(실행파일) : 실행파일을 실행시켜 인터페이스에 보여준다.

apt-get install file: file을 다운 받는다.
file (파일이름) : 해당 파일이 무엇으로 이루어져있는지 알려주는 역할을 한다.

docker 옵션
run -p (포트번호) : 포트
 -it : gcc같은 한번 실행하면 끝나는 프로세스를 제어하기 위한 옵션
-d : 백그라운드로 실행

컨테이너 접속
exec : 옵션으로 설정한 컨테이너 접속
attach : 컨테이너에 그대로 접속
